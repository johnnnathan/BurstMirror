A valid solution for this challenge according to the creator is to patch the program so that the health increases by 10, instead of decreasing, when the "Enter" key is pressed. Inside the main function we can find `SUB dword ptr [ESP + 1c], 0xa`. Inside ghidra we can right click the operation and patch it. Instead of using the "SUB" operation we will use the "ADD" operation. We can then export the program and execute it. We will see that in our version the health value goes up.
