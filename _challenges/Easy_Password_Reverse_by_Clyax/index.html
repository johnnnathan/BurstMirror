Easy Password Reverse by Clyax

📝 Solution

The program asks for a username and a password from the user. The ACII values of the characters that make up the username are summed together and the result stored. The password is parsed as a number and that means that non-numeric passwords will be registered as 0. The sum of the characters of the username must be equal to the number given as the password for the user to be granted access. In the case of a successful combination a "Good" message is shown. Example: Username: a - Password: 97 Username: { - Password: 123
