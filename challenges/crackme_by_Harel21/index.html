The program accepts a username and a password and then performs some mathematical operations upon the username before comparing the result to the password. Depending on the length of the username the operations will be different. We could use a debugger to find the correct password of a given username but we can easily copy the code into python and create a keygen, since it is mostly unobfuscated.

Example:
  Username: abc
  Password: 2

def username_handler(username):
    length = len(username)
    length_int = length

    if (length_int - 1) % 2 == 0:
        length_int = length_int - 1
        if length_int > 2:
            length_int = (ord(username[2]) * 0x240e) // 2 + 1
    else: 
        length_int = (length_int - 1) * 0x9d9fd
        if length_int < 0:
            length_int += 3 
        length_int = ((length_int >> 2) * 0xd) // 7 

    return length_int


username = input("Enter your username: ")
password = username_handler(username)
print(f"The correct password is {password}")
